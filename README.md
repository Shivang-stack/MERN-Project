# MERN-Project
## Tech Stack: 
Mongodb ,Express.js, React.js & Node.js
CSS: Bootstrap

To start 
```
>>cd projfrontend
>>npm install
after completion..
>>npm start
```
similarly in another terminal
```
>>cd projbackend
>>npm install
after completion..
>>npm start
```
